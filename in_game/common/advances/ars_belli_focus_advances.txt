# Advances gated by the second age focus chosen in ars_belli_age_focus.1.
# Pattern:
#   - No "for = adm|dip|mil" (that field hooks the engine's preference slot;
#     we're using a script-side gate instead).
#   - allow = { has_variable = ars_belli_age_focus var:ars_belli_age_focus = flag:<focus> }
#     blocks the advance until the focus is picked. The has_variable guard
#     prevents "var = flag:X" from spuriously matching pre-event countries.
#   - Replicate per age by changing "age = age_X_*" and the modifiers below.
############################

############################
